### Verification Process Applied
Each manufacturer entry was verified for:

1. **Phone Number Accuracy**
   - Number matches official manufacturer listing
   - International format validation (+1 for US/CA, etc.)
   - Active/working number status

2. **Department Routing**
   - Confirms routing to technical support department
   - Notes any IVR menu navigation required
   - Distinguishes between residential/commercial lines where applicable

3. **Metadata Quality**
   - Source URL documented
   - Verification date recorded (YYYY-MM-DD format)
   - Department type clearly identified
   - Country code specified

4. **Special Cases**
   - Multiple support centers documented separately
   - Regional variations noted
   - Specialty tags added where relevant
